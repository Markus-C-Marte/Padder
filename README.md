# Padder - Launchpad Controller

A Python application for controlling and managing launchpad MIDI devices.

## Overview

Padder provides an interface for working with Novation Launchpad controllers, allowing button assignment, LED control, and MIDI event handling.

## Features

- Launchpad device detection and initialization
- Button assignment and mapping
- LED feedback control
- MIDI event processing

## Project Structure

- `main.py` - Main application logic
- `launchpadMethods.md` - API documentation for launchpad methods

## Status

This project is in development. See `launchpadMethods.md` for implementation details.

## Technologies

- Python 3
- MIDI (Python-midi)
